# Multi-Table Merger

Author: Daniel Rudaev (D1DX)

Synchronizes records across multiple Airtable tables.

## Scripts

- `sync-all-records.js` - Full sync of all records between tables
- `sync-changes-only.js` - Incremental sync of changed records only

## Usage

Interactive extension scripts for keeping multiple tables in sync.

## Setup

1. Create an Airtable extension in your base
2. Copy desired script from `src/` folder
3. Paste into extension code editor
4. Configure source and target tables
5. Map fields between tables
6. Run the extension

## Features

- Full table synchronization
- Incremental change detection
- Field mapping configuration
- Data validation and error handling
