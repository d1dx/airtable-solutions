# SelectCase Formatter

Author: Daniel Rudaev (D1DX)

Reformats select field options with standardized naming conventions.

## Usage

Extension script that applies consistent formatting to single and multiple select field options.

## Setup

1. Create an Airtable extension in your base
2. Copy `src/selectcase-formatter.js`
3. Paste into extension code editor
4. Select target table and select fields
5. Choose formatting style
6. Run the extension

## Features

- Standardize option naming across select fields
- Apply consistent case formatting
- Bulk update field options
- Preview changes before applying
