# Record Exists Checker

Author: Daniel Rudaev (D1DX)

Checks if records exist based on specified criteria.

## Usage

Extension script for validating record presence in tables.

## Setup

1. Create an Airtable extension in your base
2. Copy `src/record-exists-checker.js`
3. Paste into extension code editor
4. Configure table and search criteria
5. Run the extension

## Features

- Query records by field values
- Validate record existence
- Return matching record details
- Support for complex search criteria

Version: 1.0.0
