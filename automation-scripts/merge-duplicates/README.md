# Merge Duplicates

Author: Daniel Rudaev (D1DX)

Merges duplicate records and consolidates data automatically.

## Usage

Interactive script that identifies and merges duplicate records based on specified criteria.

## Setup

1. Copy `src/merge-duplicates.js`
2. Create automation in Airtable
3. Add "Run a script" action
4. Paste the code
5. Configure matching criteria and merge rules
6. Test in a copy of your base first

## Features

- Identifies duplicates by field matching
- Consolidates data from duplicate records
- Deletes or archives merged records
- Preserves data integrity
